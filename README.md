# ahnaf-zaman.github.io

This repository hosts a data-driven personal portfolio built using the design document as a reference. All content lives in `assets/data.js` and the site is served via GitHub Pages.

## Editing the portfolio

1. Open `assets/data.js` and modify the fields. Add projects, skills, experience, etc.
2. If you need new images, place them in `assets/images/` and update the paths in the data file.
3. Commit your changes and push to `main` – GitHub Pages will rebuild automatically.

The rest of the files (`index.html`, `assets/style.css`, `assets/main.js`) implement the template and styling. You usually won't have to touch them unless you want to customize behavior or style.
